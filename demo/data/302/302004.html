1.方法定义

访问修饰符 返回值类型（void、int、float、String...）方法名(参数，参数...) {

代码...

return xx；

}

  • 访问修饰符 ：public static；
  • 返回值类型：没有返回值类型，使用void；有返回值类型，就使用 如 int；
  • 方法名：定义请点击链接Java规则 (zrxsky.top)；
  • 参数：可设置无，也可以设置有。
  • return xx：根据需要填写，也可能不需要填写return。
public class Test {
    public static void main(String[] args) {
        
    }
    
    public static int add(int a, int b) {
        return a+b;
    }
}

 

2.方法的调用

2.1.静态方法调用其他方法

2.1.1.静态方法调用静态方法

同一类内直接调用，不同类内直接通过类名.方法名（参数表）调用。

Test1类

package zrx.test;

public class Test1 {

    public static void main(String[] args) {
        TestT11();    //静态方法直接 调用类内 的静态方法
        Son.SonS1();  //静态方法通过类名直接 调用Son类 中的静态方法
    }

    public static void TestT11(){  //静态方法
        System.out.println("调用了，静态方法 TestT11");
    }

}

Son类

package zrx.test;

public class Son {
    public static void SonS1() {  //Son类里的静态方法
        System.out.println("S1调用了，静态方法 SonS1");
    }

    public void SonS2() {  //Son类里的非静态方法
        System.out.println("S2调用了，静态方法 SonS2");
    }

    public static void SonS3() {  //Son类里的静态方法
        System.out.println("S3调用了，静态方法 SonS3");
    }

    public void SonS4() {  //Son类里的非静态方法
        System.out.println("S4调用了，静态方法 SonS4");
    }
}

结果：

调用了，静态方法 TestT11
S1调用了，静态方法 SonS1

Process finished with exit code 0

 

2.1.2.静态方法调用非静态方法

无论是否在同一类内，均需要通过对象调用。

Test2类

package zrx.test;

public class Test2 {

    public void TestT21(){  //非静态方法
        System.out.println("调用了，非静态方法 TestT21");
    }

    public static void main(String[] args) {
        Test2 son1=new Test2();
        son1.TestT21();     //静态方法通过对象 调用类内 中的非静态方法
        Son son=new Son();  //静态方法通过对象 调用Son类 中的非静态方法
        son.SonS2();
    }

}

Son类

package zrx.test;

public class Son {
    public static void SonS1() {  //Son类里的静态方法
        System.out.println("S1调用了，静态方法 SonS1");
    }

    public void SonS2() {  //Son类里的非静态方法
        System.out.println("S2调用了，静态方法 SonS2");
    }

    public static void SonS3() {  //Son类里的静态方法
        System.out.println("S3调用了，静态方法 SonS3");
    }

    public void SonS4() {  //Son类里的非静态方法
        System.out.println("S4调用了，静态方法 SonS4");
    }
}

结果：

调用了，非静态方法 TestT21
S2调用了，静态方法 SonS2

Process finished with exit code 0

 

2.2.非静态方法调用其他方法

2.2.1.非静态方法在同一类内调用其他方法

Test3类

package zrx.test;

public class Test3 {

    public void TestT31(){  //非静态方法
        System.out.println("调用了，非静态方法 TestT31");
    }
    public static void TestT32(){  //静态方法
        System.out.println("调用了，静态方法 TestT32");
    }
    public void alloutput(){
        TestT31();  //非静态方法直接 调用类内 的非静态方法
        TestT32();  //非静态方法直接 调用类内 的静态方法
    }
    public static void main(String[] args) {
        Test3 test3 = new Test3();
        test3.alloutput();  //前边已学静态方法通过对象调用非静态方法
    }

}

结果：

调用了，非静态方法 TestT31
调用了，静态方法 TestT32

Process finished with exit code 0

 

2.2.2.非静态方法在不同类之间调用其他方法

Test4类

package zrx.test;

public class Test4 {

    public void output(){
        Son.SonS3();  //通过类名直接 调用Son类 中的静态方法，不建议使用对象调用静态方法
        Son son = new Son();
        son.SonS4();  //通过对象调 用Son类 中的非静态方法
    }

    public static void main(String[] args) {
        //同一类中的静态方法调用非静态方法output，前边已经涉及到
        Test4 test4 = new Test4();
        test4.output();
    }

}

Son类

package zrx.test;

public class Son {
    public static void SonS1() {  //Son类里的静态方法
        System.out.println("S1调用了，静态方法 SonS1");

    }

    public void SonS4() {  //Son类里的非静态方法
        System.out.println("S4调用了，静态方法 SonS4");
    }
}

结果：

Process finished with exit code 0
