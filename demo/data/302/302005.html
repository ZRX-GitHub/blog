1、顺序结构

按顺序一步一步向下执行。

 

2、选择结构

按一定的条件进行执行（不重复执行）。

if

if (条件) {
	语句;
}
else if (条件) {
    语句;
}
...
else {
    
}

条件：true、a>2、a==2……

其中，if 后面的语句可根据需要曾减。

switch

switch(条件) {
	case 常量1: 语句；break；
	case 常量2: 语句；break；
	case 常量3: 语句；break；
	...
	default: 语句;break;
}

条件：true、a>2、a==2……；

条件：如a；

常量1：如1；

其中，case的顺序不是必须按顺序的，但建议按顺序；

break根据需要增减。

3、循环结构

按一定的条件进行执行（重复执行）。

 

for

for (表达式1;表达式2;表达式3) {
	表达式4;
}

PS：表达式4为语句。

表达式1、2、3的执行顺序：

表达式1>>表达式2>>表达式3

表达式1>>表达式4>>表达式2>>(满足：执行表达式3 -> 产生新 -> 表达式4。不满足：退出for循环)

满足：表达式4>>表达式2>>(满足：执行表达式3 -> 产生新-> 表达式4。不满足：退出for循环)

……

 

while

while (条件) {
	语句;
}

条件：true、a>2、a==2……；条件为退出循环的要求。

while 与 do while的区别，while循环次数>=0（先判断条件，不满足要求，则无法进入循环），do while循环次数>0（先进入循环，再进行条件判断）。

do while

do {
	语句;
}while (条件);

条件：true、a>2、a==2……；条件为退出循环的要求。
