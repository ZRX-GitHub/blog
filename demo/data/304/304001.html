JavaScript 简介

JavaScript 是互联网上最流行的脚本语言，这门语言可用于HTML和web，更可广泛用于服务器、PC、笔记本电脑、平板电脑和智能手机等设备。

JavaScript 是脚本语言

JavaScript 是一种轻量级的编程语言。

JavaScript 是可插入 HTML 页面的编程代码。

JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。

JavaScript 很容易学习。

JavaScript：对事件的反应

浏览器将弹出一个窗口，窗口内的内容为：“ 欢迎! ”

如若需要查看，请点击下方框内的红字区域。

 <button type="button" onclick="alt('欢迎!')"> 点我! </button>

JavaScript：改变 HTML 内容

x=document.getElementById("demo");  //查找元素(ID)
x.innerHTML="Hello JavaScript";     //这里填写你点击之后改变的内容

如若需要查看，请点击下方框内的红字区域。

你好！



function myFunction() {
	x=document.getElementById("demo_1");	//找到要替换的文本 ID
	x.innerHTML="Hello！";	                //替换的文本内容
}

JavaScript：改变 HTML 图像

点击图像，改变图像（关/开灯）

ps：如果需要不规则化地触发改变图像，请选择其它方法替换。如更改为svg图像。

单引号内填写控件的ID 


					
function changeImage() {	
	element=document.getElementById('demo_2')    
	if (element.src.match("bulbon"))
	{
		element.src="https://www.runoob.com/images/pic_bulboff.gif";
	}
	else
	{
		element.src="https://www.runoob.com/images/pic_bulbon.gif";
	}
}

JavaScript：改变 HTML 样式

改变 HTML 元素的样式，属于改变 HTML 属性的变种。

JavaScript 能改变 HTML 元素的样式。


x=document.getElementById("demo_3");    //找到要改变的元素
x.style.color="#ff0000";                //改变的样式

JavaScript：验证输入

JavaScript 常用于验证用户的输入。

如果输入的不是数字，则浏览器弹出提示窗口：提示您输入的不是数字！

 
//不允许字符中出现空格 
var x=document.getElementById("demo_4").value;    //找到要判断的位置
	if (x == "" ) {
		alert("请输入字符！");
	}
	else if (x == ""||isNaN(x)) {
		alert("您输入的不是数字！");
	}
	else {
		alert("您输入的是数字！");
	}
