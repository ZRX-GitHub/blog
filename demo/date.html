常用命令

简单的查询预计：--查询当前用户下的所以对象（表） select * from tab; -- 查询员工表的所有信息 * 表示表中的所以数据，单独查询用 逗号隔开 select * from emp; -- 查询表中的个别类型的数据 select ename,sal from emp;
多表查询

多表查询：-- 笛卡尔积、子查询、连接查询（内连、满外连、左连、右连） -- 笛卡尔积 -- --比较简单、性能比较差，因为先会做排列组然后再进行条件过滤 select * from emp; select * from dept; -- 查询所有的员工信息包含部门信息 select e.*, d.* from emp e, dept d where e.deptno = d.deptno; -- 子查询 -- -- 实际开发的时候永远不要用 " * " select e.* ,（select d.dname from dept d where d.deptno = e.deptno）as dname from emp e;
where和having的区别

类型：“where”是一个约束声明，在查询数据库的结果返回之前对数据库中的查询条件进行约束，即在结果返回之前起作用，且“where”后面不能写“聚合函数”。
